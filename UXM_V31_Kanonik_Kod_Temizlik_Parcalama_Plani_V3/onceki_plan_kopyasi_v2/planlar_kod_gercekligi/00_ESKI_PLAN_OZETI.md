# 00_ESKI_PLAN_OZETI - AKTIF DEGIL, SADECE KAYIT

Bu belge, eski belgelerden kalan hedefleri tek sayfada toplar. Bu belge kod otoritesi degildir.

Eski hedefler:

1. UXM dilinden x64 ASM uretmek.
2. ASM'den OBJ ve EXE ciktisi almak.
3. Meta servisleri 1-120 araliginda genisletmek.
4. Matrix servislerini 160-176 araliginda eklemek.
5. Decimal UX-FP servislerini 200-224 araliginda eklemek.
6. Polinom turev/integral, sayisal turev ve integral servislerini 240-254 araliginda eklemek.
7. VS Code eklentisi ile build, trace, memory watch ve JSON rapor almak.
8. Interpreter ve compiler hatlarini ayri ama izlenebilir tutmak.

Eski sorunlar:

1. Plan belgeleri koddan daha hizli cogaldi.
2. .bas dosyasina markdown/sohbet metni karisti.
3. Matrislerde VAR yazilan yerlerin bir kismi test/build kaniti tasimadi.
4. Build ciktisi kaynak klasorleriyle karisti.
5. VS Code eklentisi, final/full/native exe hatlari kirikken tamam gibi gorundu.

Bu belge degistirilmeyecek. Yeni calisma `01_YENI_PLAN_KOD_GERCEKLIGI.md` ile yurutulecek.
