# UX-MINIMA x64 V3.1 Full Final Real (Calisilan)

Bu klasor, UX-MINIMA V3.1 icin aktif calisma kopyasidir.
Odak: calisan derleyici hatti, runtime, testler ve VS Code eklentisi.

## Hedef
- Derleyiciyi calisir durumda tutmak
- IDE eklentisini calisir durumda tutmak
- Tasarim belgeleri ile kod gercekligini eslemek

## Ana Klasorler
- compiler/: Derleyici kaynaklari
- runtime/: Runtime katmani
- final/: ARGE/final derleyici denemeleri
- ide/: VS Code eklentisi ve IDE entegrasyonu
- tests/: UXM test dosyalari
- build/: Derleme ara ciktilari
- math_extensions/: Matematik uzantilari
- tools/: Yardimci arac kaynaklari

## Hizli Baslangic
- build_all.bat: Compiler derleme
- run_tests.bat: Test akisi
- ide/uxminima-vscode: Eklenti kaynaklari
